# jzEditable

A jQuery plugin to add editable divs to your webpage!

## Usage

Save on blur:
```
$('.textDiv').jzEditable();
```

Save Button:
```
$('.textDiv').jzEditable({hasSaveButton: true});
```

The textarea width is set at a default of '100%'.
To change the width:
```
$('.textDiv').jzEditable({divWidth: '50%'});
```
